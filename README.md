# Bank Customer Churn Executive Dashboard & Analytics

##Project Overview
This data science project focuses on analyzing retail banking customer churn to uncover critical structural risks and propose data-driven retention strategies for executive management. 

##Key Insights & Discoveries
- **Overall Churn:** 20.4% of the customer base has left the bank, creating an urgent need for loyalty initiatives.
- **Age Profile:** Customer churn drastically peaks between the ages of 40 and 50, representing a severe loss of premium capital.
- **Geographic Anomaly:** Germany suffers from a massive 32.4% churn rate compared to France and Spain (~16%).
- **Product Vulnerability:** Customers holding 3 or 4 bank products showcase a near-100% churn rate, indicating toxic cross-selling campaigns.

##Tech Stack & Tools
- **Language:** Python
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
- **Environment:** JupyterLab / Jupyter Notebook

##Repository Structure
- `Bank_Customer_Churn_Analysis.ipynb`: Complete exploratory data analysis (EDA) and visualization code.
- `Bank_Customer_Churn_Prediction.csv`: The core banking dataset used for this analytics.
- `Bank_Customer_Churn_Analysis.pdf`: Full downloadable executive report.
